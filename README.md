# NormSEO

Парсер ключевых слов и оптимизатор текста, который работает по двум сценариям и под шесть платформ сразу — целиком в браузере, без сервера и API-ключей.

## Что умеет

**Два режима работы:**

1. **По теме** — вводите тему или ключевую фразу, получаете подобранные ключи и запросы аудитории (информационные, коммерческие, long-tail, сравнения, вопросы).
2. **По тексту** — вставляете готовый текст, получаете метрики, разбор и подсказки, какие ключи стоит добавить.

**Шесть платформ (можно выбрать несколько):**

| Платформа | Под что оптимизирует |
|-----------|----------------------|
| 🔍 **SEO** | Поисковая выдача Google / Яндекс: плотность ключа, переспам, long-tail, вопросы «Люди также спрашивают» |
| 🤖 **GEO** | Ответы ИИ (ChatGPT, Perplexity, AI Overviews): определения, факты, списки, FAQ, разговорные запросы |
| 💼 **ATS** | Резюме и вакансии: распознавание hard skills, глаголы достижений, оцифрованные результаты |
| 🎵 **TikTok** | Алгоритм рекомендаций: хуки первых секунд, ключи в подписи, нишевые хэштеги |
| 📸 **Instagram** | Reels и SEO подписей: хуки, CTA, хэштеги |
| 📌 **Pinterest** | Визуальный поиск: ключи в описании, названия досок, сезонные модификаторы |

## Как запустить

Никакой сборки не нужно — это статический сайт на чистом HTML/CSS/JS.

```bash
# просто откройте файл
open index.html

# или поднимите локальный сервер
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Как это работает

Весь анализ выполняется локально в браузере — ничего не отправляется на сервер.

- **Извлечение ключей** — токенизация, n-граммы (1–3 слова), частотность и плотность с учётом русских и английских стоп-слов.
- **Генерация запросов** — комбинирование темы с наборами модификаторов (вопросы, коммерческие, информационные, сравнения, платформенные).
- **Разбор текста** — эвристики под каждую платформу: переспам, наличие определений и цифр, словарь навыков для ATS, хуки и CTA для соцсетей.

## Структура

```
index.html              разметка и UI
assets/css/styles.css   стили (яркий, современный, лаконичный)
assets/js/data.js       стоп-слова, модификаторы, словари, конфиг платформ
assets/js/engine.js     движок анализа и генерации
assets/js/app.js        интерфейс и связка
```

## Ограничения

Инструмент использует лингвистические эвристики и словари, а не данные о реальной частотности из Wordstat / Keyword Planner. Он подсказывает структуру и направления, а объёмы запросов сверяйте с профильными сервисами.
